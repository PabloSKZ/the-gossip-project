# The Gossip Project

Gossip App in Ruby on Rails

## Installation 

### Configuration

In the terminal :

`git clone https://github.com/PabloSKZ/the-gossip-project.git`

`cd the-gossip-project`

`bundle install`

### Database creation

`rails db:create`

`rails db:migrate`

### Database initialisation

`rails db:seed`

## Connexion

`rails server`

Open a browser :

http://localhost:3000/

Have fun !
